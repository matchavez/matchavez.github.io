# Tumu Environment Planning

_Readme for the DevOps Project title page._

This DevOps project seeks to plan and display the Systems and Environment planning for the deployments related to the Tumu projects. The go-live will be an operational planning task, while the environment planning is to ensure systems and environments are optimised from development to post-deployment support.

Please view [the wiki](https://dev.azure.com/synlaitops/Tumu-environment-planning/_wiki/wikis/Deployment%20Strategies/5/Tumu-Environment-Strategies-Wiki) for supporting documentation, and the boards for known status. This project will change, and is not intended to be a source of truth for the _actual_ environments. The work here is with the context of _planning_. Thus, if a plan for a resource is completed, it may appear done here, while it is still not operationally complete.

> **Please do not confuse the "definition of done" in this project with actual operational readiness.**
