# Jira Cloud for OT / combined instance

---

## Pros for joining the IT cloud:

- One location for the company, one instance to be billed.
- Projects can be kept separate now in the same way as it has been.
- Better rate per user.


#### Other Pros for moving to the cloud
- No local maintenance.
- Availability outside VPN.
- App is usable.
- Virtually no downtime.
- Potential to use Confluence in conjunction.




## Cons

- Control of account creation ceded.


#### Other Cons for moving to the cloud

- Customisations done to the local install may not be available in the Cloud.
- Projects must be imported or recreated.
