# Clearwater

**Clearwater Golf Club** · 45 Clearwater Avenue · Par 72. Championship resort course (Sir Bob Charles design; hosts the NZ Women's Open). Each card shows the club's official hole-by-hole flyover map, which carries the definitive per-tee yardages (Black/Blue/White/Gold) and the bunker/water layout. The matrix tee distances and clubs are from a scorecard database; trust the map for exact numbers.

**Phone** 03 360 2146 · **Web** www.clearwatergolf.co.nz
